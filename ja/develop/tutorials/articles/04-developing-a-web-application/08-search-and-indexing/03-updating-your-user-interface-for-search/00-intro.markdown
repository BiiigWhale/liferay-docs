# 検索用のユーザーインターフェイスの更新

[TOC levels=1-4]

ゲストブックポートレットの検索用ユーザーインターフェイスを更新するには、2つの手順が必要です。

1.  ゲストブックポートレットのデフォルトビューJSPを更新して、クエリを送信するための検索バーを表示します。

2.  ゲストブックポートレット用の新しいJSPを作成して、検索結果を表示します。

まず、ゲストブックポートレットのビューJSPを更新します。
