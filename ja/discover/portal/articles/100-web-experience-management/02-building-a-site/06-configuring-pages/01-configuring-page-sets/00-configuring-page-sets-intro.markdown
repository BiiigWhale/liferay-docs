# ページセットの構成

[TOC levels=1-4]

ページセット全体のオプションを構成するには、 *ページ*ページセットの横にある *構成* アイコンを選択します。 ページセット用に構成されたオプションは、そのすべてのページに適用されます。 ページセットオプションは、サイトレベルで設定されたオプションをオーバーライドし、個々のページのカスタマイズは、ページセットのオプションをオーバーライドします。
